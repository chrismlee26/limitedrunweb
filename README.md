# Limited Run Admin Dashboard

v0.2.0

- Built in React, React-Router, CSS, Node.js

- Design: Figma

## Project

- This full-stack capstone project is an admin panel for project creation for my startup project: 'Limited Run'.

- Product development isn't easy. v0.2.0 is a set of tools that will help makers organize thoughts to make better decisions. These plans are shared to buyers who can view on a mobile application: [Mobile Github](https://github.com/chrismlee26/LimitedRunMob)

- the project uses the Formik Library for forms with Yup for Validation Schema

- Real Time Database will be updated in the next release

## Dependencies & Run

Type <npm install> for dependencies

Type <yarn start> to view application
